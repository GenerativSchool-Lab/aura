# Hospital Emergency Multimodal Assistant

A multimodal AI assistant designed to help hospital emergency departments with patient triage, information retrieval, and decision support.

## Architecture

### Backend
- **Framework**: FastAPI (Python)
- **Hosting**: Railway
- **AI Model**: Mistral (multimodal)
- Handles multimodal input processing, AI model integration, and API endpoints

### Frontend
- **Language**: TypeScript
- **Hosting**: Vercel
- Provides user interface for emergency staff to interact with the assistant

## Features

- Multimodal input support (text, voice, images)
- Real-time emergency triage assistance
- Patient information retrieval
- Clinical decision support

## Deployment

- Backend deployed on Railway
- Frontend deployed on Vercel
